# dotfiles-public

## Bash Prompt

Gitリポジトリ内ではブランチ名と状態が表示されます。

```
user:~/repo (main|*+%) $
```

| 記号 | 意味 |
|------|------|
| `*` | 未ステージの変更あり |
| `+` | ステージ済みの変更あり |
| `%` | 未追跡ファイルあり |
| `$` | stashあり |
| `<` | リモートより後ろ（pullが必要） |
| `>` | リモートより先（pushが必要） |
| `<>` | リモートと分岐 |
| `=` | リモートと同期済み |
